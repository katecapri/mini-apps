Мини-игра, в которой требуется цветные блоки в рамке разместить под указанный в верхней строке цвет. 
Сперва нужно нажать на цветной блок, затем с правой стороны окна выбрать направление, куда его нужно переместить.
Если перемещение невозможно, высплывает окно с ошибкой. 
Если попытаться выбрать направление без предварительно выбранного блока, всплывает окно с ошибкой. 
На черные области "наступать" нельзя.
Под выбором направлений есть кнопка сброса выбора одного из блоков (как ход отмена выбора не засчитывается).
В конце игры выводится окно с количеством передвижений и затраченным временим на прохождение. 
В папке dist находится .exe файл с игрой.
